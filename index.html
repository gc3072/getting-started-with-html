Basics of the Web and HTML

World Wide Web

Also known as “the web” or “www” could be defined as a group of computers connected to each other using applications called browsers to request web pages over HTTP. (Hyper-Text Transfer Protocol) These pages are written in HTML. (Hyper Text Markup Language)

HTML

HTML is made of:
- Text content (What you see)
- Markup (What it looks like)
- References to other documents (eg- videos and images)
- Links to other pages (URLs)

These HTML docs are grouped together or linked by URLs. (Uniform Resource Locator) These are known as the global addresses of documents and other resources in the World Wide Web.Some examples of the types of files that can be found on the web are:
Plain text
HTML
Images
Videos
Music

HTML Vocabulary

• Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag (< p > for example) and a closing tag, (< /p >). Some tags (called "void" tags) do not require a closing tag (like the < br > tag).

• Element: An HTML element refers to everything within a set of opening and closing tags. There are examples of inline tags and block tags. You can use these to manage whitespace. Inline tags just creates space around that line and does not do much else like creates a new line for text. Block creates whitespace around the content so there will be differences in how that text is displayed by the browser.

Basic HTML Markup examples:
- < b > contents < /b >
b- bold
- < em > contents < /em >
em tag- emphasis – italics
- < br > - break
Note- break is an example of a void tag where there is no additional ending tag.
- < p > - paragraph
< p > content < /p >

HTML Attributes
< tag ATTR=”value” > contents < /tag >
Anchor- < a > -link
Example- < a href=www.reddit.com > derp < /a >
note- href=attribute name and value is www.reddit.com

Images- void tag
< img > -images
< img src=”URL” alt=”text” >
Note- The alt=”text” portion is required. It will not break anything, but it is the text the browser will display for broken requests and is used for the blind, so good habit to always add it.
